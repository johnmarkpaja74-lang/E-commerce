# E-Buy E-Commerce App

E-Buy is a mobile e-commerce application built with Expo and React Native.  
It includes product browsing, category filtering, product details, cart and checkout flow, profile management, authentication, and local persistence for key user data.

## Tech Stack

- Expo + React Native
- Expo Router (file-based navigation)
- Firebase Authentication / Firestore / Storage
- SQLite for local storage
- Zustand for state management

## Run Locally

```bash
npm install
npx expo start
```
